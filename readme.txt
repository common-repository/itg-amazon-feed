Create your own amazon product feed quick and easy. 


== Description ==

If you participate in Amazon Feed Program then use this plugin to create a fully-compliant plugin.
Take full control over your website images by harnessing our powerful and easy-to-use options. Upload, focus, zoom and resize are all done effortlessly, in a matter of seconds.

== Installation ==

= Minimum Requirements =

* WordPress 4.5 or greater
* PHP version 5.4 or greater
* MySQL version 5.0 or greater

= Recommended requirements =

* PHP version 7.0 or greater
* MySQL version 5.6 or greater
* WP Memory limit of 64 MB or greater (128 MB or higher is preferred)

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Amazon Feed Products > Feed categories and create as many product categories as you require. Each category will represent a single article that will be displayed by amazon.
4. Go to Amazon Feed Products and create as many products as you need to and set their Amazon Product Properties as required by the specification. Assign products to single/multiple feeds.
5. Go to pages and create the empty page. Assign this page the Amazon Feed template and now your Amazon Feed will be accessible from the URL of this page.
